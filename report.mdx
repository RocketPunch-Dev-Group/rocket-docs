---

부적절한 콘텐츠를 발견하면 신고할 수 있고, 특정 사용자를 차단해 관련 콘텐츠를 덜 보이게 할 수 있습니다.

## 신고하기

신고할 대상의 \[⋯\] 버튼을 눌러 신고할 수 있습니다. 확인 창에서 \[신고하기\]를 누르면 접수되고, 신고한 콘텐츠는  내 화면에서 보이지 않게 됩니다. 접수된 신고는 운영팀의 내부 검토를 거쳐 처리됩니다.

| 대상 | 절차 |
| --- | --- |
| 게시물 신고 | 게시물 오른쪽 위의 \[⋯\] 버튼을 누르고 \[게시물 신고\]를 선택합니다. |
| 프로필 신고 | 신고할 사용자의 프로필 화면에서 \[⋯\] 버튼을 누르고 신고를 선택합니다. |
| 메시지 신고 | 상대와의 채팅 화면에서 \[⋯\] 버튼을 누르고 신고를 선택합니다. |
| 디스커버 신고 | (\*수정) |

## 차단하기

특정 사용자가 불편하다면 차단할 수 있습니다. 상대의 프로필, 상대가 쓴 게시물·댓글, 상대와의 채팅에서 \[⋯\] 버튼을 눌러 차단합니다. 게시물에서는 오른쪽 위 \[⋯\] → \[프로필 차단\]을 누르면 됩니다.

차단하면 이렇게 바뀝니다.

- 홈 피드와 허브에서 상대의 콘텐츠를 추천하지 않습니다.
- 상대의 활동 알림을 받지 않습니다.
- 팔로우 중이었다면 팔로우가 해제됩니다.
- 서로 메시지를 주고받을 수 없습니다.

## 차단 해제하기

설정 → `소셜 네트워킹` → `차단한 프로필`에서 차단한 사용자를 확인하고 해제할 수 있습니다.
