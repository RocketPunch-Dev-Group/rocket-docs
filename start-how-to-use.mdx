## 원하는 내용을 빠르게 찾는 법

<CardGroup cols={3}>
  <Card title="검색으로 찾기" icon="magnifying-glass">
    화면 위쪽 검색창에 궁금한 내용을 입력하세요. 단축키 Ctrl K(또는 ⌘ K)로 바로 열 수 있습니다.
  </Card>

  <Card title="AI에게 물어보기" icon="sparkles">
    `Ask AI`를 누르고 궁금한 점을 문장으로 물어보면, 가이드 내용을 바탕으로 답변을 정리해 줍니다.
  </Card>

  <Card title="메뉴로 찾기" icon="bars">
    좌측에서 버티컬 서비스별로 사용법을 찾아볼 수 있습니다.<br />
  </Card>
</CardGroup>

## 가이드를 읽을 때 알아두면 좋은 점

<Tip>
  **화면에 보이는 이름 그대로 안내해요**<br />버튼이나 메뉴 이름은 실제 사이트에 보이는 이름 그대로 적었습니다. 예: \[채용\], `간편 지원하기`
</Tip>

## 다른 도움이 필요하다면?

<Tip>
  가이드에서 답을 찾지 못하셨다면, 지원센터로 문의해 주세요.
</Tip>
