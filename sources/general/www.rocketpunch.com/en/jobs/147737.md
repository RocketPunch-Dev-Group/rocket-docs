Team Introduction

📋 Here are the tasks we will collaborate on! - Frontend development of the renewal of the social media automation solution Modigram - Frontend development of the renewal of the home-based work service Viewup - Improvement of code quality and refactoring of projects in development or already developed

Responsibilities

💡 We are looking for someone to join our frontend team that develops Smile Dragon's core services! 🌏 We work in an environment like this! - Communication within the development team is done through Slack. - Pull Requests must undergo code reviews from team members. - Project progress and information are managed through Notion. - Development environments are unified through Docker. - Project builds and deployments are automated through GitHub Actions. - Deployments are done in an internal Kubernetes + Helm + Argo CD environment. 🛠️ We use technologies like these! - Next.js, TypeScript, Emotion, Zustand, React Query - Yarn Berry, pnpm, Rollup.js, Jest, Cypress You can check out more about our tech stack at https://stackshare.io/companies/smiledragon 🙋 We want to work with someone like this! - Someone with 5 or more years of frontend development experience or equivalent skills - Someone with a basic understanding of HTML5, CSS, JavaScript, and TypeScript - Someone with development experience using React and Next.js - Someone with design, development, deployment, and operational experience in at least one frontend project - Someone proficient with version control systems using Git and GitHub - Someone who enjoys finding better code or development methods - Someone who can make decisions based on an understanding of business pace ☀️ It would be even better if you’re like this! - Someone with SSR development experience using Next.js - Someone with frontend performance optimization and test code writing experience - Someone who has experience styling using CSS-in-JS methods like Emotion and styled-components - Someone with development experience in a Docker environment - Someone who has experienced deployment cycles through CI/CD - Someone with experience improving responsive web, SEO, web accessibility, and web standards - Someone with component-driven development experience using Storybook 😎 Benefits and welfare - 80 minutes for lunch: take your time to eat and use the remaining time to relax or handle personal matters! - Convenient vacation system: you can apply for annual leave a day in advance and half-day leave on the same day! - Book purchasing support: from now on, the company buys books! - Ananti Penthouse (6-star hotel): available for everyone to use. - Free lunch: save about 2.4 million won annually. - Work equipment support: necessary equipment is provided promptly! - Personal cabinet provided: keep important items safe! - Support for celebrations and condolences: Smile Dragon is with you in both happy and sad times. - Birthday benefits: enjoy delicious cake and leave 2 hours early on your birthday. - Fast-track promotion system based on evaluation: rewards you for your hard work! - Cafeteria operation: coffee, chat, etc. are available anytime during work hours! - 4 major insurances and retirement pension system: standard benefits 💗 Plus!
