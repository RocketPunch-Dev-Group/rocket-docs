# Getting Started

From App Key issuance to your first API call — get started in 3 steps.

🚧

Guide details will be added soon

Currently in invite-only builder beta — only the core flow is covered.

## 1\. Register an App & Get a Key

Log in with your Rocketpunch account, register an App, and get the right credentials instantly.

[Get an App Key](https://developers.rocketpunch.com/apps/new)

## 2\. Set Up Auth Headers

Include your issued App Key as a header in every request.

`X-RP-API-Key: rp_app_********`

## 3\. Make Your First API Call

Check the endpoint list and sample requests in the API docs.

[View API Docs ↗](https://developers.rocketpunch.com/docs/index.html?lang=en 'Opens in a new tab')
