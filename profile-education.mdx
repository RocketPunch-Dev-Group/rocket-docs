## 학력 등록하기

학교뿐만 아니라 수료한 교육과정, 부트캠프 등도 등록할 수 있습니다.

| 항목 | 내용 |
| --- | --- |
| 교육기관 | 학교 또는 교육기관명입니다. 검색해서 선택하거나 직접 입력할 수 있습니다. |
| 유형 | 고등학교, 전문학사, 학사, 석사, 박사, 교육과정 중에서 선택합니다. |
| 전공/학부 | 검색해서 선택하거나 직접 입력합니다. |
| 상태 | 졸업, 재학, 수료, 휴학, 중퇴 중에서 선택합니다. |
| 재학 기간 | 시작일과 종료일을 선택합니다. |
| 관련 기술 | 배운 기술을 대표 5개까지 등록할 수 있습니다. |
| 설명 | 전공 또는 이수한 교육을 설명합니다. 최대 2,000자까지 입력할 수 있습니다. |
