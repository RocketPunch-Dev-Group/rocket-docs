## 라이브러리 채우기

라이브러리에는 이력서, 포트폴리오, 자기소개서처럼 **나를 더 잘 소개할 수 있는 파일이나 URL**을 등록할 수 있습니다. 라이브러리는 프로필에서 확인할 수 있습니다. 라이브러리에 

### 파일 추가하기

- 유형(이력서, 포트폴리오, 자기소개서, 직접 입력)을 고르고 파일을 첨부합니다. 
- hwp, hwpx, doc, docx, ppt, pptx, xls, xlsx, pdf, key 형식을 지원하며 최대 100MB까지 첨부할 수 있습니다.

### URL 추가하기

- 유형을 고르고 URL을 입력합니다. 
- 이력서 · 포트폴리오 · 자기소개서 외에 LinkedIn, Github, Instagram, Threads, X, Facebook 같은 소셜 유형도 선택할 수 있습니다.

## 라이브러리 공개 범위 설정

라이브러리에 등록한 pdf나 링크는 항목마다 공개 범위를 따로 설정할 수 있습니다.

| 공개 범위 | 내용 |
| --- | --- |
| 전체 공개 | 로켓펀치의 모든 회원이 볼 수 있습니다. |
| 채용담당자에게만 공개 | 내게 제안을 보낼 수 있는 인사담당자나 헤드헌터가 볼 수 있습니다. |
| 비공개 | 공고에 지원할 때만 사용하며, 다른 사용자는 볼 수 없습니다. |
