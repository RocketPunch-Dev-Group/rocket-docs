## 경력 등록하기

경력을 등록하면 다른 사용자에게 노출되고, 새로운 제안을 받을 확률이 높아집니다.<br />

| 항목 | 내용 |
| :-- | :-- |
| 소속 | 기업 또는 소속 단체명입니다. 검색해서 선택하거나 직접 입력할 수 있습니다. |
| 고용형태 | 정규직, 계약직, 위촉직, 인턴, 프리랜서, 자영업 중에서 선택합니다. |
| 역할 | 직책 또는 직무입니다. |
| 근무 기간 | 시작일과 종료일을 선택합니다. 현재 재직 중이라면 종료일 없이 등록할 수 있습니다. |
| 관련 기술 | 사용한 기술을 대표 5개까지 등록할 수 있습니다. |
| 설명 | 담당한 직책 또는 업무를 설명합니다. 최대 2,000자까지 입력할 수 있습니다. |

한 기업에서 맡은 여러 직무는 각각 경력으로 추가할 수 있습니다. 현재 재직 중인 경력이 가장 먼저 보이고, 이후 시작일 기준 최신순으로 정렬됩니다.
