로켓펀치 Open API는 아시아 최대 규모의 IT 인재 네트워크에 대한 프로그래밍 방식 접근을 제공한다. 채용 공고 애그리게이터, 채용 플랫폼, 커리어 분석 도구 등 무엇을 만들든, API는 채용 공고·기업·프로필·이벤트에 대한 실시간 데이터를 단일 인증 엔드포인트와 일관된 다국어 스키마 뒤에서 노출한다.

## 무엇을 만들 수 있는가

개발자와 프로덕트 팀은 로켓펀치 Open API를 다양한 활용 사례에 사용한다:

- **채용 공고 애그리게이터** — 키워드, 위치, 카테고리로 필터링한 실시간 채용 공고를 가져와 자체 제품 안에 노출한다.
- **채용 도구** — 로켓펀치 네트워크의 실시간 기업·후보자 프로필 데이터로 ATS나 소싱 워크플로를 보강한다.
- **인재 분석** — 한국 IT 생태계 전반의 채용 트렌드, 스킬 수요, 기업 성장 신호를 분석한다.
- **커리어 앱** — 실시간 시장 데이터를 기반으로 개인화된 채용 추천 엔진이나 커리어 경로 탐색기를 만든다.
- **기업 리서치 도구** — 기업 프로필, 인원 신호, 채용 공고를 결합해 투자자용 또는 세일즈 인텔리전스 대시보드를 만든다.

## 작동 방식

<Steps>
  <Step title="앱 등록">
    [developers.rocketpunch.com/apps/new](https://developers.rocketpunch.com/apps/new)로 이동해 로켓펀치 계정으로 로그인하고, 앱 이름과 설명을 입력한다. App Key가 즉시 발급되며 — 별도의 승인은 필요하지 않다.
  </Step>
  <Step title="요청 인증">
    `X-RP-API-Key` 헤더를 사용해 모든 API 요청에 App Key를 추가한다. 모든 엔드포인트는 동일한 base URL을 공유한다: `https://openapi.rocketpunch.com`.
  </Step>
  <Step title="엔드포인트 호출 및 출시">
    필요한 엔드포인트를 쿼리하고, 응답 데이터를 제품에 통합하고, 개발자 대시보드에서 실시간 사용량을 모니터링한다. 앱을 로켓펀치 커뮤니티에 공유할 준비가 완료된다.
  </Step>
</Steps>
