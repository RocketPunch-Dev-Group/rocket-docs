이 가이드는 앱 등록, 요청 인증, 로켓펀치 Open API에서 첫 응답 읽기 과정을 안내한다. 5분 안에 실시간 데이터가 프로젝트로 흘러들어오게 된다.

## 단계

<Steps>
  <Step title="앱 등록">
    [developers.rocketpunch.com/apps/new](https://developers.rocketpunch.com/apps/new)로 이동해 로켓펀치 계정으로 로그인한다. 다음 필드를 입력한다:

    | 필드 | 설명 |
    | --- | --- |
    | **앱 이름** | 프로젝트를 알아볼 수 있는 짧은 이름 |
    | **설명** | 앱이 무엇을 하고 로켓펀치 데이터를 어떻게 사용하는지 |
    | **콜백 URI** | _(선택)_ 사용자 동의 흐름에 OAuth 2.0을 사용할 계획일 때만 필요 |

    **등록**을 클릭한다. App Key가 즉시 발급된다 — 검토 기간도, 대기도 없다.

    <Note>
      App Key는 `rp_app_xxxxxxxxxxxxxxxx` 형식을 따른다. 지금 복사해 안전한 곳에 보관하고, 비밀번호처럼 다룬다.
    </Note>
  </Step>
  <Step title="첫 API 호출하기">
    `X-RP-API-Key` 요청 헤더에 App Key를 추가하고 아무 엔드포인트나 호출한다. 아래 예시는 키워드 `AI`에 해당하는 채용 공고를 검색한다:

    <CodeGroup>

    ```javascript JavaScript
    const res = await fetch(
      "https://openapi.rocketpunch.com/api/v1/jobs?keyword=AI",
      { headers: { "X-RP-API-Key": "rp_app_YOUR_KEY_HERE" } },
    );
    const { items } = await res.json();
    console.log(items);
    ```

    ```python Python
    import requests
    
    response = requests.get(
        "https://openapi.rocketpunch.com/api/v1/jobs",
        params={"keyword": "AI"},
        headers={"X-RP-API-Key": "rp_app_YOUR_KEY_HERE"},
    )
    data = response.json()
    print(data["items"])
    ```

    ```bash cURL
    curl -X GET 'https://openapi.rocketpunch.com/api/v1/jobs?keyword=AI' \
      -H 'X-RP-API-Key: rp_app_YOUR_KEY_HERE'
    ```

    </CodeGroup>

    <Tip>
      `rp_app_YOUR_KEY_HERE`를 등록 시 복사한 App Key로 바꾼다.
    </Tip>
  </Step>
  <Step title="응답 확인하기">
    성공한 요청은 `items` 배열이 담긴 JSON 객체를 반환한다. 배열의 각 요소는 하나의 채용 공고를 나타낸다:

    ```json
    {
      "items": [
        {
          "id": "123456",
          "title": "AI Research Engineer",
          "company": {
            "id": "78910",
            "name": "Example Corp",
            "slug": "example-corp"
          },
          "location": "Seoul, Korea",
          "tags": ["AI", "Machine Learning", "Python"],
          "posted_at": "2025-01-15T09:00:00Z",
          "url": "https://www.rocketpunch.com/jobs/123456"
        }
      ],
      "total": 842,
      "page": 1,
      "per_page": 20
    }
    ```

    | 필드 | 타입 | 설명 |
    | --- | --- | --- |
    | `items` | array | 쿼리에 일치하는 채용 공고 객체 목록 |
    | `total` | integer | 모든 페이지에 걸친 일치 결과 총 개수 |
    | `page` | integer | 현재 페이지 번호 |
    | `per_page` | integer | 페이지당 반환되는 결과 수 |

    `200 OK` 상태는 키가 유효하고 요청이 성공했음을 확인해준다. `401`을 받았다면 `X-RP-API-Key` 헤더를 포함했는지, 키가 올바른지 다시 확인한다.
  </Step>
</Steps>
