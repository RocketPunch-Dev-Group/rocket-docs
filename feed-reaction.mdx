## 게시물에 반응하기

| 버튼 | 내용 |
| --- | --- |
| **Boost** | 게시물에 공감을 표시합니다. |
| **댓글** | 게시물에 댓글을 남깁니다. |
| **재게시** | 게시물을 내 피드에 다시 올립니다. |
| **공유** | 게시물 링크를 복사합니다. |

## 부적절한 게시물을 발견했다면?

게시물의  더보기 버튼 `⋮` 을 누르면 해당 게시물을 신고하거나 작성자를 차단할 수 있습니다. 신고 절차와 처리 과정은 [게시물 신고하기](/report-post) 를 참고하세요.
