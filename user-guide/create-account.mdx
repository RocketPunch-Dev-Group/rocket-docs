로켓펀치 계정을 만들면 수천 개의 IT 채용 공고, 기업 프로필, 네트워킹 이벤트, 교육 프로그램에 즉시 접근할 수 있어요 — 모두 무료예요. 가입은 2분이 채 안 걸리고, 이메일 주소만으로 시작할 수 있어요. 가입을 마치면 전문가 프로필을 만들고 바로 한국 IT 생태계와 연결되기 시작해요.

## 로켓펀치 가입하기

<Steps>
  <Step title="로켓펀치에 접속해 로그인 페이지 열기">
    [rocketpunch.com](https://www.rocketpunch.com)에 접속해 페이지 오른쪽 위의 **로그인**을 클릭하세요. [rocketpunch.com/en/login](https://www.rocketpunch.com/en/login)로 바로 이동해도 돼요.
  </Step>
  <Step title="이메일 주소 입력하고 계속하기">
    이메일 주소를 입력하고 **계속**을 클릭하세요. 로켓펀치는 이메일 기반 인증을 사용하기 때문에 처음부터 따로 비밀번호를 만들 필요가 없어요.
  </Step>
  <Step title="이메일 인증하기">
    받은편지함에서 로켓펀치가 보낸 인증 메일을 확인하세요. 메일을 열어 링크를 따라가거나 인증 코드를 입력해 주소를 인증하고 계정을 활성화하세요.
  </Step>
  <Step title="기본 프로필 완성하기">
    이메일 인증이 끝나면 이름, 헤드라인, 몇 가지 핵심 정보를 입력하라는 안내가 나와요. 이 정보를 채우면 프로필이 검색에 노출돼요. 더 많은 정보는 언제든 나중에 추가할 수 있어요 — 자세한 방법은 [프로필 설정하기](/user-guide/profile-setup)를 참고하세요.
  </Step>
</Steps>

## 로그인 옵션

로켓펀치는 **이메일 기반 로그인**을 사용해 인증을 간단하고 안전하게 유지해요. 등록한 이메일 주소를 입력하고 인증 단계를 완료하면 바로 로그인돼요. 별도의 서드파티 OAuth 로그인은 필요하지 않아요. 계정을 만들면 로켓펀치의 [이용약관](https://www.rocketpunch.com/en/terms)과 [개인정보처리방침](https://www.rocketpunch.com/en/privacy)에 동의하는 것으로 간주돼요. 이 문서들은 여러분의 데이터가 플랫폼에서 어떻게 저장되고 사용되는지 설명해요.

## 언어 설정

로켓펀치는 **11개 언어**를 지원해, 한국 시장에서 일하거나 한국 시장과 협업하는 해외 IT 전문가도 편하게 사용할 수 있어요. 플랫폼 기본 언어는 한국어지만, 언제든 영어나 다른 지원 언어로 바꿀 수 있어요.

표시 언어를 바꾸려면:

1. 오른쪽 위의 프로필 아바타를 클릭하세요.
2. **설정**을 여세요.
3. **언어** 항목에서 원하는 언어를 드롭다운에서 선택하세요.
4. 변경 사항을 저장하면 인터페이스가 즉시 업데이트돼요.

영어 사용자는 [rocketpunch.com/en](https://www.rocketpunch.com/en)으로 바로 접속해 영어 모드로 시작할 수도 있어요.

<Tip>
  가입 직후 프로필을 완성하면 리크루터에게 노출될 가능성이 크게 높아져요. 사진, 헤드라인, 경력 한 건 이상, 스킬 목록이 있는 프로필은 기업·리크루터 검색에 훨씬 자주 나타나요. [프로필 설정하기](/user-guide/profile-setup)에서 시작해 보세요.
</Tip>

## 계정 보안

로켓펀치 계정에 연결된 이메일 주소에 강력하고 고유한 비밀번호를 사용하고, **설정 → 보안**에서 활성 세션을 주기적으로 확인해 계정을 안전하게 지키세요. 로그인 정보를 공유하지 말고, 로켓펀치를 사칭하는 피싱 메일을 조심하세요 — 공식 메일은 항상 `@rocketpunch.com` 주소에서 발송돼요.

계정에 로그인할 수 없거나, 무단 접근이 의심되거나, 계정 복구가 필요하면 [support@rocketpunch.com](mailto:support@rocketpunch.com)으로 지원팀에 직접 문의하세요.

<Note>
  가입을 완료하면 로켓펀치의 [이용약관](https://www.rocketpunch.com/en/terms)과 [개인정보처리방침](https://www.rocketpunch.com/en/privacy)에 동의하는 것으로 간주돼요. 계정을 만들기 전에 두 문서를 모두 자세히 확인할 수 있어요.
</Note>
