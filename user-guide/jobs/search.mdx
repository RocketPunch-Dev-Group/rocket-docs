로켓펀치는 한국의 스타트업과 대기업 생태계 전반에 걸친 수천 개의 IT 기회와 여러분을 연결해요. 빠르게 성장하는 스타트업에서 React 직무를 찾는 프론트엔드 엔지니어든, 대규모 인프라 팀을 노리는 노련한 DevOps 전문가든, 채용 검색 페이지는 잡음을 걷어내고 실제로 원하는 조건에 맞는 포지션을 찾아주는 도구를 제공해요.

## 키워드 검색

[채용 공고 페이지](https://www.rocketpunch.com/en/jobs) 상단의 검색창이 출발점이에요. 직무 제목, 회사명, 기술 스킬은 물론 복지와 혜택까지 검색하기 때문에, 넓게든 구체적으로든 원하는 만큼 검색할 수 있어요.

원하는 직무나 근무 환경을 설명하는 어떤 단어든 입력하면, 로켓펀치가 그 단어를 공고 어디에든 언급한 목록을 보여줘요.

**쉼표로 여러 키워드 검색하기**

쉼표로 구분하면 한 번에 여러 단어를 검색할 수 있어요. 로켓펀치는 쉼표로 구분된 각 단어를 독립적인 키워드로 처리해, 그중 하나라도 일치하는 공고를 보여줘요 — 더 넓게 찾거나 여러 조건을 한 번의 쿼리로 조합할 수 있어요.

예를 들어 다음을 입력하면:

```text
Next.js, PowerPoint, 중식 제공, 대출이자 지원, 성과급
```

…Next.js 스킬을 언급하거나, PowerPoint를 요구하거나, 중식 제공·대출이자 지원·성과급 같은 혜택을 제공하는 공고를 보여줘요. 기술뿐 아니라 여러분에게 가장 중요한 혜택으로도 쉽게 필터링할 수 있어요.

<Tip>
  `React`, `Kubernetes`, `TypeScript` 같은 구체적인 스킬 키워드를 사용하면, 넓은 카테고리 필터에만 의존하지 않고 직무 설명에 해당 기술을 명시한 공고를 찾을 수 있어요.
</Tip>

## 정렬 옵션

키워드와 필터를 설정했다면 정렬 메뉴로 결과가 표시되는 순서를 조절하세요.

| 정렬 옵션 | 동작 |
| --- | --- |
| **관련도순** | 검색 키워드와 얼마나 밀접하게 일치하는지로 공고를 정렬해요. 구체적인 검색어를 입력했을 때 가장 좋아요. |
| **인기순** | 후보자의 관심을 가장 많이 받은 공고를 보여줘요. 어떤 직무가 화제인지 가늠할 때 유용해요. |
| **최신순** | 가장 최근에 올라온 공고를 먼저 보여줘요. 기업이 새로 올리는 공고를 놓치지 않을 때 사용하세요. |
| **기업 규모순** | 결과를 기업 규모 큰 순(또는 그 반대)으로 정렬해요. 기업 규모가 최우선 고려 사항일 때 유용해요. |

<Note>
  로켓펀치의 채용 공고는 채용하는 기업에 따라 한국어 또는 영어로 게시돼요. 주로 영어로 읽는다면 영어 설명이 포함된 공고를 찾거나, 플랫폼의 이중 언어 인터페이스를 사용해 공고를 탐색하세요.
</Note>
