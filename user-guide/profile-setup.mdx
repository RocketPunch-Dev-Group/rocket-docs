로켓펀치 프로필은 플랫폼에서 여러분을 나타내는 공개 전문가 신원이에요. 경력, 기술 스킬, 학력, 프로젝트를 하나의 페이지로 모아, 기업과 리크루터가 후보자를 찾을 때 검색하게 돼요. 잘 만든 프로필은 여러분의 이야기를 전할 뿐 아니라, 검색에 잘 노출되게 만들어요. 프로필이 완성도 높고 구체적일수록, 한국 IT 업계 채용 담당자의 관련 검색에 더 자주 노출돼요.

## 프로필 섹션

프로필은 6개의 핵심 섹션으로 구성돼요. 해당하는 항목을 최대한 많이 채우세요 — 각 섹션이 깊이를 더하고 검색 결과에서의 노출을 높여줘요.

| 섹션 | 포함할 내용 |
| --- | --- |
| **기본 정보** | 이름, 프로필 사진, 전문 헤드라인, 지역, 짧은 자기소개 |
| **경력** | 회사명, 직함, 기간, 그리고 무엇을 만들거나 이끌었는지에 대한 설명이 담긴 과거·현재 직무 |
| **스킬 & 기술** | 사용하는 프로그래밍 언어, 프레임워크, 도구, 플랫폼 (예: React, Python, Kubernetes) |
| **학력** | 학위, 학교, 전공, 졸업 연도 |
| **프로젝트 & 포트폴리오** | 개인 프로젝트, 오픈소스 기여, 출시한 제품, 또는 작업물 링크 |
| **자격증** | 여러분 분야와 관련된 전문 자격증, 라이선스, 인증 |

## 단계별로 프로필 만들기

<Steps>
  <Step title="프로필 페이지로 이동하기">
    [rocketpunch.com](https://www.rocketpunch.com)에 로그인하고 오른쪽 위의 아바타를 클릭하세요. 드롭다운에서 **프로필**을 선택해 프로필 편집기를 여세요.
  </Step>
  <Step title="경력 추가하기">
    **경력 추가**를 클릭해 직함, 회사명, 재직 기간, 그리고 담당 업무와 성과에 대한 짧은 설명을 입력하세요. 관련 직무마다 항목을 하나씩 추가하세요. 사용한 기술과 이뤄낸 성과를 언급하며 구체적으로 작성하세요.
  </Step>
  <Step title="스킬과 기술 스택 나열하기">
    **스킬** 섹션에 사용하는 프로그래밍 언어, 프레임워크, 데이터베이스, 도구를 추가하세요. 각 스킬을 입력한 뒤 자동완성 목록에서 선택해, 플랫폼의 색인 용어와 일치하도록 하세요. 여러분의 전문성을 정확히 나타내는 8~15개의 스킬을 목표로 하세요.
  </Step>
  <Step title="학력 정보 추가하기">
    **학력 추가**를 클릭해 학교명, 학위 종류, 전공, 재학 기간을 입력하세요. 전문가로서의 이야기를 강화한다면 관련 수강 과목, 수상, 학술 프로젝트도 포함하세요.
  </Step>
  <Step title="프로젝트나 포트폴리오 연결하기">
    **프로젝트** 섹션에 주목할 만한 작업물의 제목, 설명, 링크를 추가하세요 — GitHub 저장소, 실제 서비스 중인 제품, 디자인 포트폴리오, 발표한 논문 등 무엇이든 좋아요. 프로젝트는 여러분이 무엇을 만들 수 있는지 리크루터에게 구체적으로 보여줘요.
  </Step>
</Steps>

<Tip>
  **React**, **Python**, **Kubernetes**, **TypeScript**, **AWS** 같이 구체적이고 검색 가능한 기술 스킬을 추가하면 리크루터 검색에 노출될 가능성이 바로 높아져요. 로켓펀치의 리크루터는 기술 스택으로 후보자를 자주 필터링하니, 실제 업무에서 쓸 도구에 맞춰 스킬을 나열하세요.
</Tip>

## 프로필 공개 범위

기본적으로 로켓펀치 프로필은 **공개** 상태라, 기업, 리크루터, 다른 플랫폼 회원이 발견할 수 있어요. 공개 범위 설정은 언제든 바꿀 수 있어요:

- **공개** — 전체 프로필이 모든 로켓펀치 사용자에게 보이고 검색 결과에 나타날 수 있어요. 기회를 열어두고 있다면 권장하는 설정이에요.
- **비공개** — 프로필이 검색 결과에서 숨겨지고 다른 사용자에게 보이지 않아요. 계정을 삭제하지 않으면서 플랫폼 활동을 잠시 멈추고 싶을 때 사용하세요.

공개 범위를 변경하려면 **설정 → 개인정보**로 이동해 프로필 공개 설정을 전환하세요.

<Warning>
  프로필을 비공개로 설정하면 리크루터 검색에서 완전히 제외돼요. 적극적으로 구직 중이라면 발견될 가능성을 극대화하기 위해 프로필을 공개로 유지하세요.
</Warning>

## 핸들(사용자명)

모든 로켓펀치 프로필에는 고유한 **핸들**이 있어요 — 개인 프로필 URL을 이루는 짧은 사용자명이에요. 예를 들어 핸들이 `yourname`이면 URL은 `rocketpunch.com/yourname`이 돼요. 핸들은 가입 시 설정되며 **설정 → 계정**에서 변경할 수 있어요.

전문적이고 공유하기 쉬운 핸들을 고르세요 — 많은 사용자가 자기 이름이나 그 변형을 사용해요. 핸들은 공개 프로필, 검색 결과, 그리고 리크루터나 협업자와 공유하는 직접 링크에 나타나요. 핸들 정책과 사용에 대한 자세한 내용은 로켓펀치의 [이용약관](https://www.rocketpunch.com/en/terms)을 참고하세요.
