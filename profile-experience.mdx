## 활동 등록하기

참가, 수상, 자격증, 프로젝트, 논문/저서, 뉴스/미디어, 특허 등 커리어를 위해 진행한 활동을 등록할 수 있습니다. 활동을 등록하면 더 많은 제안을 받을 수 있습니다.

| 유형 | 내용 | 날짜 | 관련 회사·기관 | 함께한 사람 |
| --- | --- | --- | --- | --- |
| 참가 | 행사 · 컨퍼런스 · 이벤트 참가 내역 | 참가일/시작일 · 종료일(장기 일정 행사 선택 가능) | ✓ | ✓ |
| 수상 | 대회 · 수상명 | 수상일 | ✓ | ✓ |
| 자격증 | 자격증명 | 취득일 | - | - |
| 프로젝트 | 프로젝트명(투자 유치 경험 포함) | 시작일 · 종료일(진행 중 선택 가능) | ✓ | ✓ |
| 논문/저서 | 논문 · 저서명 | 발행일 | - | ✓ |
| 뉴스/미디어 | 뉴스 · 미디어 제목(인터뷰 포함) | 방영 · 게재일 | - | - |
| 특허 | 특허명 · 특허등록번호 | 출원일 | ✓ | ✓ |

모든 유형에 공통으로 설명 또는 URL과 관련 기술(대표 5개까지)을 입력할 수 있습니다. '함께한 사람'은 로켓펀치 프로필을 언급해 연결할 수 있습니다.
