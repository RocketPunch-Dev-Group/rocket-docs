로켓펀치는 2012년에 시작한 비즈니스·커리어 네트워킹 플랫폼입니다. 스타트업 정보를 한곳에서 찾기 어렵던 시기에, 흩어진 정보를 모아 보여주는 실험적인 블로그에서 출발했습니다. 현재 서비스는 주식회사 더블에이스가 운영합니다.

## 연혁

| 시기 | 내용 |
| --- | --- |
| 2012년 9월 | 베타 서비스를 공개했습니다. 스타트업 정보를 모아 보여주는 서비스로 시작했습니다. |
| 2016년 4월 | 서비스를 사람 중심으로 전면 개편했습니다. 기업 정보를 모아 보여주는 구조에서, 개인 프로필을 중심에 두는 구조로 바꿨습니다. |
| 2017년 12월 | 프로필 10만 개를 넘었습니다. |
| 2019년 9월 | 프로필 20만 개를 넘었습니다. |
| 2022년 9월 | 서비스 10주년을 맞았습니다. |

## 무엇이 달라졌나요

로켓펀치는 스타트업 정보를 모아 보여주는 서비스로 시작했습니다. 이후 기업보다 사람을 중심에 두는 방향으로 바뀌었고, 개인 프로필을 기반으로 채용·네트워킹·이벤트를 연결하는 플랫폼으로 확장됐습니다.

지금의 로켓펀치는 여러 서비스가 합쳐져 하나를 이룹니다. 각 서비스가 무엇을 하는지는 [로켓펀치는 어떤 서비스인가요?](/start-introduction) 에서 확인할 수 있습니다.
