로켓펀치는 봇이나 가짜 계정이 아닌 실제 사용자를 위한 환경을 만들기 위해 World ID 인간 인증을 지원합니다. 이 문서에서는 World ID가 무엇이고, 로켓펀치에서 어떻게 인증하는지 안내합니다.

## World ID가 무엇인가요?

World ID는 World가 제공하는 인간 인증 수단입니다. 이름이나 연락처 같은 다른 정보를 공개하지 않고, 봇이나 가짜 계정이 아닌 인증된 고유 사용자임을 증명할 수 있습니다. 한 사람이 하나의 계정을 사용하는지 확인하는 [계정 인증하기](/account-identity-verification)와는 별개로, World ID는 계정 사용자가 실제 인간임을 증명하는 수단입니다.

로켓펀치에서 인간 인증을 완료하면 더 많은 프로필과 게시물, 대화에 접근할 수 있습니다.

## World ID 등록하기

내 프로필에서 인증하기를 누르면 World ID 인증 화면이 열리고 World 앱으로 인증을 진행합니다.

World ID 인증은 인증된 고유 사용자인지 확인하는 장치인 Orb에서 진행됩니다. Orb에서 인증을 마치면 인증된 World ID 정보가 World App에 저장되어, 로켓펀치를 비롯한 온라인 서비스에 로그인할 때 사용할 수 있습니다.

인간 인증을 완료하면 인증 데이터는 암호화되어 사용자 휴대폰으로 전송되고, Orb에는 남지 않습니다.

## World ID 인증 정보는 로켓펀치가 관리하지 않아요

World ID는 오픈 소스 기반의 탈중앙화, 비허가형 프로토콜 위에 구축되어 있어, 누구의 소유도 아니며 누구나 이를 연결하고 상호작용할 수 있는 방식을 구축할 수 있습니다. World ID와 인증 데이터는 사용자 휴대폰에만 저장되므로, 사용자가 직접 관리합니다.

자세한 내용은 [World 홈페이지](https://world.org/ko-kr/world-id)를 참고하세요.
