로켓펀치는 봇이나 가짜 계정이 아닌 실제 사용자를 위한 환경을 만들기 위해 World ID 인간 인증을 지원합니다. 이 문서에서는 World ID가 무엇이고, 로켓펀치에서 어떻게 인증하는지 안내합니다.

## World ID가 무엇인가요?

World ID는 World가 제공하는 인간 인증 수단입니다. 이름이나 연락처 같은 다른 정보를 공개하지 않고, 봇이나 가짜 계정이 아닌 인증된 고유 사용자임을 증명할 수 있습니다. 한 사람이 하나의 계정을 사용하는지 확인하는 [계정 인증하기](/account-identity-verification)와는 별개로, World ID는 계정 사용자가 실제 인간임을 증명하는 수단입니다.

로켓펀치에서 인간 인증을 완료하면 더 많은 프로필과 게시물, 대화에 접근할 수 있습니다.

## World ID로 로켓펀치에서 인간 인증하기

내 프로필에서 `인증하기`를 누르면 `World App으로 로그인`을 선택하여 인간 인증을 완료할 수 있습니다. 완료되면 로켓펀치 프로필에 인간 인증이 표시됩니다. 아직 World ID가 없다면 아래에서 발급 방법을 확인할 수 있습니다.

## World ID 인증 정보는 로켓펀치가 관리하지 않아요

Orb에서 인증을 마치면 인증 데이터는 암호화되어 사용자 휴대폰으로 전송되고, Orb에는 남지 않습니다. World ID와 인증 데이터는 사용자 휴대폰에만 저장되므로 사용자가 직접 관리합니다.

World ID는 특정 회사가 소유하지 않는 오픈 소스 기반의 탈중앙화 프로토콜 위에 만들어졌습니다. 로켓펀치도 인증 과정에서 World ID 정보를 저장하거나 관리하지 않습니다.

자세한 내용은 [World](https://world.org/ko-kr/world-id)를 참고하세요.
